# Contributing

Thanks for your interest! This project is open source and PRs are very welcome.

## Adding a Template

Templates are the easiest and most-needed contribution — I'm not a designer (hence the AI), so the template library leans on the community.

1. Open the editor and create a beautiful design.
2. Use **File → Export** to download the JSON, and export a PNG preview.
3. Create a new folder under [public/templates/](public/templates/) and drop both files in:
   ```
   public/templates/<your-template-name>/
     <your-template-name>.json
     <your-template-name>.png
   ```
4. Open a PR. No code changes needed — the Templates sidebar auto-discovers folders.

## Code Changes

For bug fixes and features:

1. Fork the repo and create a branch.
2. Follow the [Setup](README.md#setup) steps to run locally.
3. Open a PR describing what changed and why.

## Reporting Bugs

Open an issue with reproduction steps and (if possible) a screenshot or screen recording.
